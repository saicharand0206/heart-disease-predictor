# ❤️ Heart Disease Prediction App

This project is a Deep Learning-based web application that predicts the risk of heart disease based on patient medical data. The model is trained using clinical parameters and deployed using Streamlit for an interactive user interface.

## 🚀 Live Demo
👉 https://cherrychills-heart-disease-predictor-v3.hf.space

## 🧠 Project Overview
Heart disease is one of the leading causes of death worldwide. This application helps in early prediction by analyzing key health indicators such as age, blood pressure, cholesterol, and more.

The model takes user input and predicts whether the person is at high risk or low risk of heart disease.

## ⚙️ Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Streamlit
- Hugging Face Spaces (for deployment)

## 📊 Input Features
The model uses the following parameters:
- Age
- Sex (0 = Female, 1 = Male)
- Chest Pain Type (0–3)
- Resting Blood Pressure
- Cholesterol Level
- Fasting Blood Sugar (>120 mg/dl)
- Rest ECG (0–2)
- Max Heart Rate
- Exercise Induced Angina
- Oldpeak
- Slope (0–2)
- Number of Major Vessels (0–3)
- Thal (0–3)
